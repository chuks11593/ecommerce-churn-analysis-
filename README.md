# ecommerce-churn-analysis-
A 4-table relational SQL analysis that cleans and transforms raw sales data into a month-over-month net revenue report for legacy inventory performance.

Inventory Clearance Analysis:
Problem: The marketing team needed to identify revenue generated from older stock (pre-2018) to plan clearance sales.

Action: Developed a 4-table join query to calculate actual revenue after discounts, resolving data type conflicts in the legacy database.

Result: Provided a clean, month-over-month revenue report for all historical legacy sales.
